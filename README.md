# 📊 HR Dashboard - Power BI Project

## 🏆 Overview
This Power BI dashboard provides key insights into employee data, including:
- **Employee Attendance & Performance**
- **Contract & Position Details**
- **Department-wise Analysis**
- **HR Order Management**
